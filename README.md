# 活俠傳 繁體中文＋日本語 雙語字幕 MOD

*Legend of Mortal — Traditional Chinese / Japanese bilingual subtitle mod (fan-made, unofficial)*

一個讓《活俠傳 Legend of Mortal》全程顯示「繁體中文＋日本語」雙語字幕的非官方玩家自製 MOD。

《活俠傳》是臺灣獨立遊戲團隊「原始鳥熊 Obb Studio Inc.」開發的單機角色扮演遊戲。
🎮 遊戲本體 Steam 商店頁面：https://store.steampowered.com/app/1859910/Legend_of_Mortal/?l=tchinese

## ⬇️ 下載與安裝 / ダウンロードとインストール

**[👉 點此下載最新版 ZIP](https://github.com/enishiyui520/legend-of-mortal-bilingual-mod/releases/latest)**

1. 打開《活俠傳》的安裝資料夾（能看到 `Mortal.exe` 的那一層）。
   例：`C:\Program Files (x86)\Steam\steamapps\common\LegendOfMortal`
2. 把下載的 ZIP **全部內容**解壓到該資料夾，跟 `Mortal.exe` 放在同一層（不要多包一層資料夾；若之前裝過舊版直接覆蓋即可）。
3. **【重要】把遊戲語言設成「简体中文」**（畫面顯示的文字仍然是繁體，這只是 MOD 掛載機制需要的底層設定）：
   - 方法 A：雙擊解壓出來的「設定簡中語言.bat」
   - 方法 B：進遊戲 → 設定 → 語言 → 选「简体中文」
4. 啟動遊戲即可。

**日本語でのインストール手順：**
1. 『活俠傳』のインストールフォルダ（`Mortal.exe` があるフォルダ）を開く。
2. ダウンロードしたZIPの中身を**すべて**そのフォルダに展開（フォルダを二重にしない。旧バージョンがある場合はそのまま上書きでOK）。
3. 【重要】ゲーム内言語を「简体中文」に設定してください（表示される文字自体は繁体字のままです。これはMOD側の仕組み上必要な設定です）。
   - 方法A：展開したフォルダの「設定簡中語言.bat」をダブルクリック
   - 方法B：ゲーム内 → 設定 → 言語 → 「简体中文」を選択
4. ゲームを起動すれば完了です。

### 🎛️ 遊戲中熱鍵 / ゲーム内ホットキー

| 熱鍵 | 作用 |
|---|---|
| `F7` / `F6` | 字級 放大／縮小（不會小於預設值） |
| `F5` | 開關「文字放大＋對話框自動增高」功能 |
| `F9` | 開關「對話文字強制白色」 |
| `F8` | 除錯用，一般不需要 |

## ✨ 這個 MOD 在官方日文化 MOD 之上額外做了什麼

- **全遊戲雙語顯示**：對話、戰鬥、人物介紹、道具、技能、選項、狀態欄，全部同時顯示繁中＋日文
- **自動放大字級**＋對話框自動增高以容納文字
- **依情境上色**：一般劇情日文黃色、戰鬥畫面日文紅色，跟繁體中文（白色）一眼區分
- **狀態欄直排選單**：日文獨立左欄並排顯示，不與中文疊字

## ⚠️ 誠實講：目前已知的限制（不是完美方案）

- **雙語字幕偶爾仍會超出對話框範圍**（尤其是戰鬥畫面裡的超長句子）。已經做了自動放大上限、框自動增高、超長句自動微縮字級等處理來降低機率，但**沒有完全解決**，遇到極端長句還是可能被裁切或看起來擠。這是目前最大的已知瑕疵，如果你在意排版完美，請先有心理準備。
- 覆蓋率約 99.4%，少數 UI 文字（骰子標頭等約 400 條）目前只有日文、沒有中文對照。
- 日文為玩家有志翻譯，非官方，翻譯品質持續由原作者團隊改善中。

**日本語：**
- **バイリンガル字幕は稀にダイアログボックスからはみ出すことがあります**（特に戦闘中の長い文章）。自動拡大の上限設定・ボックスの自動拡張・長文時の自動縮小などで発生頻度を減らしていますが、**完全には解決していません**。極端に長い文章では文字が見切れたり詰まって見えることがあります。これが現状最大の既知の欠点です。
- カバー率は約99.4%、一部のUI（サイコロのヘッダー等、約400件）は日本語のみで中国語対訳がありません。
